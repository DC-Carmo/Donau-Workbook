# Rugby Game Plan – Donau Workbook

This project is a digital playbook designed to structure attacking and defensive systems in rugby.

## Features
- Game plans and structures
- Set-piece organization
- Visual diagrams
- Coaching frameworks

## Vision
To create a digital coaching tool that helps players and coaches understand rugby systems in a simple and visual way.

## Next Steps
- Interactive field visualization
- Game analysis tools
- Mobile-friendly version
